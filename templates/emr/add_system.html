About External Systems

External systems represent EMR, hospital systems, labs, or other healthcare platforms that can be integrated with 6th Sense for data exchange.

Authentication Types
  • OAuth2: Uses tokens with automatic refresh capabilities
  • API Key: Uses a single API key for authentication
  • Basic: Uses username/password authentication
  • JWT: Uses JSON Web Tokens for authentication
Tip: After creating a system, you'll need to add connection details and data mappings before you can start synchronizing data.
