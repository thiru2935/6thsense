About Connections

Connections store the authentication credentials needed to communicate with the external system. These credentials are used to securely retrieve and send data.

Security Note: All sensitive credentials are stored securely and encrypted in the database. Never share these credentials or expose them in client-side code.
After Adding a Connection

Once you've added a connection, you can test it to ensure it can properly communicate with the external system. If successful, you can then begin setting up data mappings and linking patients.
